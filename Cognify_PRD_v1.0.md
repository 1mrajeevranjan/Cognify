# **3\. Problem Statement**

Task management software in 2026 forces users to choose between mutually exclusive trade-offs:

* Platform reach vs. design quality (cross-platform apps are functional but ugly; beautiful apps are Apple-only)

* Feature completeness vs. simplicity (apps with Pomodoro \+ habits \+ AI feel overwhelming; focused apps lack basics)

* Affordability vs. capability (AI scheduling costs $19–$22/month; no affordable middle ground exists)

* Personal productivity vs. light collaboration (solo tools lack sharing; team tools are overkill for families and pairs)

* Subscription fatigue vs. one-time cost (subscriptions feel like a tax; one-time apps lack cross-platform sync)

**The result:** Users churn across apps, maintain parallel tools (task app \+ habit tracker \+ calendar \+ timer), or settle for a suboptimal experience. The average knowledge worker uses 2.4 productivity apps simultaneously — a signal of unmet need, not solved problem.

# **6\. Feature Scope — MoSCoW Framework**

Features are prioritized using the MoSCoW method: Must Have (launch), Should Have (first 3 updates), Could Have (roadmap), Will Not Have (out of scope).

## **6.1  Phase 1 — Must Have (Launch v1.0)**

*These are the features that determine whether Cognify is usable and competitive from Day 1\. Without these, the app cannot be launched.*

### **Core Task Engine**

* To-dos with title, notes (rich text \+ markdown), due date, start date (Things 3 differentiator — Todoist lacks this elegantly)

* Checklists / sub-tasks (up to 2 levels deep)

* Natural language date parsing ("tomorrow", "next Monday at 3pm", "every second Thursday") — minimum parity with Todoist

* Tags (multi-tag per task), Projects, Areas — the full GTD hierarchy from Things 3

* Priorities (P1–P3 flags \+ visual color coding)

* Repeating tasks with all standard patterns (daily, weekly, monthly, custom)

* Start date vs Due date distinction (Things 3 feature Todoist lacks; helps deferred tasks not clutter Today view)

### **Views & Navigation**

* Today view — tasks due today \+ calendar events (like Things 3 Today)

* Upcoming view — 7-day rolling timeline with calendar event overlay

* Inbox — quick capture, unsorted, process later

* Someday / On Hold list

* Logbook (completed task archive)

* Project view with Heading support (like Things 3 project structure)

* Area view (like Things 3 Areas of Responsibility)

### **Cross-Platform (Day 1 Parity)**

* iOS native app (iPhone \+ iPad)

* Android native app — this alone differentiates vs Things 3 and OmniFocus

* macOS native app

* Web app (responsive — works on Windows, Linux, Chromebook)

* Seamless cloud sync — real-time, offline-first with conflict resolution

### **Reminders & Notifications**

* Time-based reminders (user-set time alerts)

* Location-based reminders (iOS only for v1)

* Reliable push notifications (this is a \#1 complaint across all competitors — invest engineering here)

* Morning digest notification: daily briefing of Today's tasks at user-chosen time

### **Calendar Integration**

* Two-way Google Calendar sync — calendar events visible in Today and Upcoming alongside tasks

* Apple Calendar (iCal) read integration

* Outlook Calendar read integration

### **Quick Entry & Capture**

* Keyboard shortcut Quick Entry (global hotkey on macOS)

* Share Sheet integration (iOS / Android) — capture from any app

* Widget (iOS home screen \+ lock screen, Android home screen)

* Email-to-task forwarding address (à la Todoist's Mail to Things)

### **Design & UX**

* Clean, minimal, premium visual design — target Things 3-level perceived quality

* Dark mode (auto \+ manual toggle) — beautifully implemented, not afterthought

* Smooth animations and micro-interactions

* Drag-and-drop rescheduling in Upcoming view

* One-handed thumb-friendly mobile layout

### **Pricing & Monetisation (v1)**

* Generous free tier: unlimited projects, unlimited tasks, unlimited tags, 2 calendar integrations, basic reminders — no crippling limits like Todoist/TickTick

* Cognify Pro: ₹149/month or ₹1,199/year (India) / $3.99/month or $34.99/year (global) — AI features, advanced repeats, all integrations, themes

* Optional Lifetime License: ₹3,499 (India) / $59.99 (global) — for trust-building and acquisition

## **6.2  Phase 2 — Should Have (v1.1–v1.3, Months 2–6)**

*These features drive retention and differentiation. Ship within the first 3 major updates.*

### **AI Features — The Core Differentiator**

* AI Daily Planner: Each morning, AI reviews your task list, calendar events, and energy patterns to suggest a daily schedule — "Do these 5 tasks today, here's the order and why"

* Smart Task Breakdown: Type any goal and AI breaks it into actionable subtasks (similar to Todoist Assist but available in Pro, not an add-on)

* Voice-to-Task (AI): Speak naturally ("I need to prep for the board meeting next week, it's important") and AI creates a structured task with date, priority, and subtasks

* AI Scheduling Suggestions: When a task has a deadline and estimated duration, AI finds open calendar slots and suggests time blocks — user approves before locking in

* Natural language input on steroids: "Submit report every second Thursday at 3pm p1 \#work" — full NLP at Todoist-level quality

### **Focus & Productivity Suite (TickTick Gap)**

* Built-in Pomodoro timer with task integration — start a task, timer runs, session logged

* Focus Mode: Single-task view, session timer, notes, AI-generated 5-step breakdown

* Focus statistics: Daily/weekly focus time, sessions completed, streaks

* Habit tracker: Create recurring daily/weekly habits with streak tracking (TickTick has this; neither Things 3 nor Todoist do natively)

### **Light Collaboration**

* Shared Lists: Share any list or project with up to 5 people (free), unlimited (paid) — this is the Things 3 gap that loses family \+ small team users

* Task assignment within shared projects (assign to a member, see their tasks)

* Comments on tasks (async team communication)

* Activity feed for shared projects

* Link sharing: Generate a read-only link to a project (share with someone without account)

### **Deeper Integrations**

* Slack integration: Create tasks from Slack messages; daily digest to Slack

* GitHub / GitLab: Link tasks to issues / PRs (developer persona)

* Notion import: One-click import from Notion databases (capture switchers)

* Todoist import: Import all projects and tasks from Todoist (capture switchers)

* Zapier \+ Make (Integromat) support for custom automations

* Apple Shortcuts / Android Tasker support

### **Customisation**

* Custom themes (10 preset themes \+ custom accent color) — currently paywalled in TickTick

* App icon choices (iOS)

* Custom sorting and grouping rules per view

## **6.3  Phase 3 — Could Have (v2.0+, Months 7–18)**

*Roadmap features that address power users and expand addressable market.*

### **Advanced AI (Level 3–4 Intelligence)**

* Predictive task duration: AI learns how long you take on specific task types and auto-estimates duration for scheduling

* AI priority calibration: AI identifies tasks you repeatedly defer and prompts a decision ("You've pushed this 6 times — is it still relevant?")

* Meeting intelligence: Connect Google Meet / Zoom — AI extracts action items from meeting transcripts and creates tasks automatically

* Email scanning: Connect Gmail / Outlook — AI scans inbox for action items and surfaces them as suggested tasks (not auto-creates, user approves)

### **Cognify for Teams (Small Teams, Up to 25 People)**

* Team workspace with shared inbox, projects, and assignments

* Project templates library

* Workload view: See all tasks per team member to avoid overload

* Basic Kanban board view for team projects

* Team analytics: completion rates, overdue tasks by person, weekly report

### **Advanced Views**

* Kanban board for any project or tag

* Eisenhower Matrix (urgent/important 2x2) — TickTick has this but buries it

* Time-blocking calendar view: Tasks become blocks on a visual timeline (drag to reschedule)

* Weekly review mode: GTD-inspired weekly review prompts

* Custom filter builder (advanced Todoist-level query language)

### **Platform Expansions**

* Apple Watch native app with complications

* Wear OS widget (Android watch)

* Browser extension (Chrome, Firefox, Safari) — clip to task with one click

* Windows native app (not just web wrapper)

* iPad-optimised layout with split view and keyboard shortcut parity

## **6.4  Will Not Have (Out of Scope)**

* Full project management (Gantt charts, dependencies, resourcing) — this is ClickUp/Asana territory; not the target

* Document editing / note-taking beyond task notes — this is Notion territory

* CRM functionality — out of scope

* Time tracking (billable hours) — separate product category

* Code repository integration at v1 — Phase 3 only

# **11\. Technical Requirements & Architecture**

## **11.1  Core Technical Principles**

* Offline-first: All data stored locally, sync when connected — tasks must work without internet (Notion fails this test)

* End-to-end encryption option for task data (privacy-sensitive users in enterprise)

* Sub-100ms task creation and navigation on mobile — performance is a feature

* Real-time sync with conflict resolution (last-write-wins with merge for concurrent edits)

* GDPR-compliant data architecture from Day 1 (enables EU market)

# **15\. Appendix**

## **15.1  Glossary**

| Term | Definition |
| ----- | ----- |
| GTD | Getting Things Done — a personal productivity methodology by David Allen; Things 3 is considered the gold standard GTD app |
| Areas | In GTD/Things 3 terminology: top-level life domains (Work, Health, Family) that contain multiple projects and loose tasks |
| Someday | A holding bucket for tasks and projects with no committed date; part of the GTD workflow |
| NLP | Natural Language Processing — the ability to parse human-readable input like "tomorrow at 3pm" into structured date/time data |
| CRDT | Conflict-free Replicated Data Type — a data structure that enables real-time offline-first sync without conflicts |
| MRR | Monthly Recurring Revenue — the normalized monthly revenue from subscription customers |
| ARPU | Average Revenue Per User — MRR divided by total paid users |
| CAC | Customer Acquisition Cost — total marketing spend divided by new customers acquired |
| LTV | Lifetime Value — estimated total revenue from a customer over their subscription lifetime |
| ASO | App Store Optimisation — process of improving app visibility in App Store and Play Store search results |
| PPP | Purchasing Power Parity — economic adjustment that prices products relative to local purchasing power rather than currency conversion |
